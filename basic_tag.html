,information Software is a set of programs and instructions that tell a computer what to do. Main types: System software: Runs and manages the computer, e.g. Windows, Android, device drivers. Application software: Helps users perform tasks, e.g. Microsoft Word, Chrome, WhatsApp. Programming software: Used to create other software, e.g. VS Code, Python, compilers. Software is different from hardware, which is the physical part of a computer—such as the keyboard, monitor, and processor.
